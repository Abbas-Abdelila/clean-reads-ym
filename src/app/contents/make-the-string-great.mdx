### Python Implementation Iterative Method

```python
class Solution:
    def makeGood(self, s: str) -> str:
        i = 0
        while i < len(s) - 1:
            if abs(ord(s[i])-ord(s[i+1])) == 32:# we have valid pair to be deleted
                s = s[:i] + s[i+2:]
                i = 0 # back to start as new pair may araise after modifying s.
            else:
                i += 1
        
        return s
```

### 2. Solution using Stack ğŸ“š

<aside>
ğŸ’¡ The way we are going to solve this using is by using the stack to keep the characters of the string as we iterate it. 
â†’ The element on top of the stack will be the previous character. We check if the      difference between the previous character from the stack and current character from the iteration is 32 (then it is valid pair to be deleted).

â‡’ If that is the case, we pop the last element from the stack, else we append it to the stack and move on to the next iteration.

â‡’ At the end what will remain in the stack will not have consecutive same small and capital letter or the other way as we have blocked that possibility of that happening by popping when we see a potential of such order happening in the stack.

</aside>

### Python implementation of stack solution

```python
class Solution:
    def makeGood(self, s: str) -> str:
        stack = []
        
        for curr_ch in s:
            if stack and abs(ord(curr_ch)-ord(stack[-1])) == 32:
                stack.pop()
            else:
                stack.append(curr_ch)
        
        return "".join(stack)
```
